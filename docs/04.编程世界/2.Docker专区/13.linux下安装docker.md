linux下安装docker

## 1，尽量在centos7和6下面安装，8已经寿终正寝



## 2，运行如下命令

```dockerfile
yum update
yum install epel-release -y
yum clean all
yum list
yum install docker-io -y
systemctl start docker
docker -v
```
- docker即可安装成功。

## 3，设置docker开机启动
```dockerfile
systemctl enable docker
```

docker安装elasticsearch

## 1，下载镜像
- docker pull elasticsearch:2.4.4


## 2，运行镜像
- docker run -d -p 9200:9200 -p 9300:9300 --name search elasticsearch:2.4.4


## 3，查看正在运行的容器，验证开启
- docker ps

## 4.再次验证是否安装成功
- curl 127.0.0.1：9200

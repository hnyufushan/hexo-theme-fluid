docker安装nexus详细步骤


## 1，启动容器。
- 运行。

```
docker run -d -p 8081:8081 -p 8082:8082 -p 8083:8083 --name nexus3 -v /home/nexus/nexus-data:/nexus-data --restart=always sonatype/nexus3
```

## 2，docker ps查看容器。
- 查看容器

```dockerfile
docker ps
# 查看日志
docker logs -f nexus3
```

Obsidian插件周刊第二期

文章发布较早，内容可能过时，阅读注意甄别。

# 前言

请参阅：Obsidian插件周刊说明 (opens new window)。

# Obsidian Leaflet

  • 插件名称：Obsidian Leaflet
  • 插件作者：Jeremy Valentine
  • 插件说明：笔记内的交互式地图
  • 插件仓库：点我跳转 (opens new window)

一款能够添加动态地图的插件，只不过地图资源在国内很难加载出来，因此不太推荐。

# Note Refactor

  • 插件名称：Note Refactor
  • 插件作者：James Lynch
  • 插件说明：将笔记内容提取成新笔记和拆分笔记
  • 插件仓库：点我跳转 (opens new window)

笔记重构插件，能够让一篇长文，按照指定的分割标志切割成链接引用风格的文章。详情参考 (opens new window)。

# Calendar

  • 插件名称：Calendar
  • 插件作者：Liam Cain
  • 插件说明：黑曜石的简单日历小部件
  • 插件仓库：点我跳转 (opens new window)

这个插件为黑曜石创建了一个简单的日历视图，用于在你的日常笔记之间可视化和导航。详情参考 (opens new window)。

# Text expand

  • 插件名称：Text expand
  • 插件作者：MrJackphil
  • 插件说明：搜索和粘贴/转换包含已创建文件的链接
  • 插件仓库：点我跳转 (opens new window)

此插件能够将符合搜索条件的内容呈现在当前文件中。

比如我想在某文件呈现所有包含daily标签的文件，可以先定义如下语句：

```expander
daily
```

2
3

然后在命令框中运行expand命令，即可将对应文件引用在当前文件内。

更加详细的用法以及查询语句，可参考其官方文档的说明。

# Jump to link

  • 插件名称：Jump to link
  • 插件作者：MrJackphil
  • 插件说明：此插件允许使用热键打开当前文档中的链接
  • 插件仓库：点我跳转 (opens new window)

运行命令能够通过快捷键跳转到文本中的链接或者文本处：

# Reading Time

  • 插件名称：Reading Time
  • 插件作者：avr
  • 插件说明：将当前笔记的阅读时间添加到黑曜石的状态栏
  • 插件仓库：点我跳转 (opens new window)

在ob底部状态栏展示当前文档阅读时间的插件。

# Todoist Sync Plugin

  • 插件名称：Todoist Sync Plugin
  • 插件作者：jamiebrynes7
  • 插件说明：在黑曜石笔记中实现Todoist任务。
  • 插件仓库：点我跳转 (opens new window)

提供了任务管理增强功能。

官方动图如下：

注意：经论坛@Masktial (opens new window)同学提醒，此插件需要配合Todoist软件使用才行。

# Vimrc Support

  • 插件名称：Vimrc Support
  • 插件作者：esm
  • 插件说明：使用黑曜石Vim命令自动加载启动文件。
  • 插件仓库：点我跳转 (opens new window)

这个插件为ob的vim模式提供了增强，因为我没有使用，因此不做过多介绍。

# Shortcuts extender

  • 插件名称：Shortcuts extender
  • 插件作者：kitchenrunner
  • 插件说明：使用快捷方式输入特殊符号，无需语言切换
  • 插件仓库：点我跳转 (opens new window)

提供了6级标题的快捷键，以及一些其他特殊符号的快捷键。

到目前为止，已经分配了以下键盘快捷键：

  • Alt+ 1-7用于输入!, @, #, $, %, ^, ,&符号
  • Alt+ 西里尔文х或ъ（或 EN[和]）用于输入[``]
  • Alt++Shift西里尔文х或ъ（或 EN{和}）用于输入{``}
  • Alt+ 西里尔文б或ю（或 EN,和.）用于输入<``>
  • Alt+ё用于输入代码栅栏的西里尔文
  • Alt+ Shift+ 西里尔文ё（或 EN ~）用于 python 代码的输入代码栅栏
  • Alt+ Shift+\对输入|的符号

# Add links to current note

  • 插件名称：Add links to current note
  • 插件作者：MrJackphil
  • 插件说明：此插件添加了一个命令，该命令允许在所选笔记的底部添加到当前笔记的链接
  • 插件仓库：点我跳转 (opens new window)

该插件提供了将当前笔记以链接的形式添加到另一个笔记底部的功能。
